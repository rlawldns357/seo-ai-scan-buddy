# SearchTune OS 역키워드 노출 전략

## 1차 샘플 결과

일반 역키워드 12개에서 Naver 상위 HTML 내 `searchtuneos.com` 직접 노출 감지 안 됨. 브랜드/도메인 검색에서는 노출되지만, 문제형·방법형·비교형 키워드는 아직 약함.

## 색인/제목 검색 샘플

- `AEO란? AI 답변 엔진에 내 콘텐츠를 인용시키는 전략`: SearchTune 감지=True, slug hint=True
- `SEO vs AEO vs GEO 차이점 통합 전략`: SearchTune 감지=False, slug hint=False
- `FAQ 스키마로 AEO 점수 올리는 방법`: SearchTune 감지=True, slug hint=False
- `AI 크롤러 robots.txt 접근 허용 설정법`: SearchTune 감지=False, slug hint=False
- `네이버 서치어드바이저 완벽 가이드 사이트 등록 최적화`: SearchTune 감지=False, slug hint=False
- `아임웹 SEO 최적화 완벽 가이드`: SearchTune 감지=False, slug hint=False
- `카페24 쇼핑몰 SEO 완벽 가이드`: SearchTune 감지=False, slug hint=False

## 전략

1. **브랜드 키워드 방어**: SearchTune OS / 서치튠OS는 유지. 이미 Naver에서 일부 노출.
2. **답변형 키워드 공략**: 블로그 FAQ 질문을 title/H2/FAQ JSON-LD/본문 첫 문단에 더 직접 반영.
3. **페이지-키워드 매칭**: 각 핵심 글마다 타겟 키워드 3~5개를 명시하고 내부링크 앵커를 그 키워드로 연결.
4. **색인 요청 우선순위**: 브랜드 검색에 잡힌 글보다, 일반 키워드에서 미노출인 글을 Search Advisor에 우선 제출.
5. **경쟁 SERP 대응**: `SEO AEO GEO 진단`은 광고/경쟁 도메인이 강하므로 랜딩/블로그 둘 다 강화.

## 우선 보강 대상

- SEO vs AEO vs GEO 차이점과 통합 전략
- FAQ 스키마로 AEO 점수 올리는 방법
- AI 크롤러 robots.txt 접근 허용 설정법
- 네이버 서치어드바이저 완벽 가이드
- 카페24 쇼핑몰 SEO 완벽 가이드
- 아임웹 SEO 최적화 완벽 가이드
