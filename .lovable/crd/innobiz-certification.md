## 3. 핵심 기술 포지셔닝: "AI 검색 진단 엔진"

신청서·기술백서·현장평가 시 **일관되게 사용할 기술 정의문**:

> **SearchTune OS Engine:** URL 한 줄 입력만으로 **SEO(전통 검색)·AEO(AI 답변 인용)·GEO(생성형 검색 가시성)** 3개 축을 동시에 계측·채점하는 한국어 특화 진단 엔진. Google PSI 실측 + Firecrawl 크롤링 + Gemini 결정론적 추론(temperature=0)을 결합한 **하이브리드 채점 모델**과, 분기별 자동 업데이트되는 **엔진 룰북(2025 Schema.org · 네이버 웹마스터 공식 룰북 포함)** 으로 구성.

### 3.1 기술 차별성 3대 포인트 (신청서 핵심 문항용)

1. **3축 통합 채점 (Industry First)**
   - 기존 도구는 SEO 단일축. 우리는 AEO·GEO를 정량 점수화한 국내 최초급.
   - Worst Score Priority 알고리즘으로 약점 자동 노출.

2. **결정론적 AI 채점 (Reproducibility)**
   - LLM 기반인데도 `temperature=0` + 룰북 강제 주입으로 동일 URL 동일 점수 보장.
   - 일반 LLM 도구의 비재현성 한계 극복.

3. **자동 진화 엔진 (Living Rulebook)**
   - `pg_cron` 기반 월 2회 (1일/15일) 자동 업데이트.
   - Perplexity sonar-pro로 트렌드 수집 → AI 검증 → `engine_config` 테이블 갱신.
   - 네이버 공식 룰북은 보존 가드로 절대 덮어쓰기 금지.

### 3.2 기술 백서(Technical Whitepaper) 목차 — 별도 PDF로 제출 권장

```
1. 배경: AI 검색 시대의 도래와 기존 SEO의 한계
2. 시스템 아키텍처
   2.1 Frontend (React + Vite)
   2.2 Backend (Supabase Edge Functions, 20+ 함수)
   2.3 분석 파이프라인 (Firecrawl → Gemini → 점수 산출)
3. 채점 알고리즘
   3.1 SEO 신호 가중치 (메타·Canonical·Sitemap·구조화 데이터)
   3.2 AEO 신호 (Q&A 구조·FAQPage 스키마·직접 답변 가능성)
   3.3 GEO 신호 (AI 봇 robots 허용·엔티티 명확성·인용 적합도)
4. 룰북 거버넌스 (Naver Webmaster Rulebook 보존 정책)
5. 성과 지표 (분석 건수·도메인 다양성·재현율)
6. 보안·프라이버시 (RLS, IP 레이트리밋, UUID 스토리지)
7. 로드맵
```

## 5. 신청서 작성 가이드 (항목별 핵심 문구)

### 5.1 "기업 개요" 항목
> 당사는 2025년 설립된 한국어 기반 AI 검색 최적화 SaaS 기업으로, 자체 개발한 'SearchTune OS' 엔진을 통해 국내 웹사이트의 SEO/AEO/GEO 가시성을 진단·개선하는 서비스를 제공한다. 누적 [N]건의 도메인 진단 실적과 [N]편의 자동 발행 콘텐츠 운영 노하우를 보유하고 있다.

### 5.2 "핵심 기술" 항목
> SEO·AEO·GEO 3축 통합 채점 엔진. Google PageSpeed Insights 실측 데이터와 Firecrawl 크롤링, Gemini LLM의 결정론적 추론(temperature=0)을 결합한 하이브리드 모델로, 동일 URL에 대해 항상 동일한 점수를 산출하는 재현성을 확보. 분기별 자동 업데이트되는 룰북 거버넌스 시스템(`update-analysis-engine`)으로 검색 알고리즘 변화에 자동 적응.

### 5.3 "기술의 혁신성" 항목 (3개 키워드로 구조화)
1. **세계 최초급 3축 통합** — 기존 SEO 도구는 단일축, 우리는 AI 검색 시대 대응
2. **결정론적 LLM 채점** — temperature=0 + 룰북 강제 주입으로 LLM의 재현성 한계 극복
3. **Self-Updating Engine** — pg_cron 기반 자동 학습으로 별도 인력 없이 최신성 유지

### 5.4 "시장성" 항목
> 글로벌 AI 검색 시장은 2024년 ChatGPT Search, Perplexity, Google SGE 출시로 급팽창. 국내 검색 광고 시장 [N]조 규모 중 AI 검색 대응 도구 시장은 미개척 블루오션. 당사는 한국어·네이버 특화 진단으로 진입장벽 구축.

### 5.5 "기대효과" 항목
> 인증 취득 시 정책자금 우대금리로 R&D 재투자 가속, 정부조달 시장 진입, 우수 R&D 인력 채용 가점 활용으로 향후 3년 내 매출 [N]배, 고용 [N]명 확대 목표.
